# Лабораторна робота №1 студенки км 81 Верзун Поліни

## Запуск програми
Для всіх варіантів ОС треба виконати:
```
docker-compose up
python3 -m pip install virtualenv
python3 -m venv env
source env/bin/activate
source .env
python3 -m pip install -r requirements.txt
python3 main.py
```
